# Thursday Bot

Thursday Bot is a Discord bot that automates the Thursday Discord server.
It automates everything from renaming new members to "Thursday" on join to
creating and closing Thursday channels at midnight on Thursdays and Fr*days.

This bot is currently a work-in-progress bit is almost ready for use. See
[TODO.md](TODO.md) for the full list of planned features.
